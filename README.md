# 🌏 Bihar Paryatan – Tourism Website

## 📖 About the Project
**Bihar Paryatan** is a frontend web project that highlights the tourism attractions of Bihar.  
The website provides information about famous tourist places, historical sites, and cultural heritage of Bihar in an easy and visually appealing way.

The aim of this project is to promote tourism and help users explore the beauty, history, and culture of Bihar through a simple and responsive web interface.

---

## ✨ Features
✔️ Simple and clean user interface  
✔️ Information about famous tourist destinations in Bihar  
✔️ Image gallery of tourist places  
✔️ Easy navigation between pages  
✔️ Beginner-friendly frontend design  

---

## 🛠️ Technologies Used
- 🌐 HTML  
- 🎨 CSS  
- ⚡ JavaScript
